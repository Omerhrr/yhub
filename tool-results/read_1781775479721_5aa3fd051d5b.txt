   694Ôćĺ   694Ôćĺ- **Theme:** Reproduce the CSS variables exactly (see ┬ž4) ÔÇö they map 1:1 to shadcn/ui's token system. The hover color `#013756` (slightly lighter than primary `#013156`) is hard-coded on nav links ÔÇö replicate or replace with `text-primary`.

   702Ôćĺ   702Ôćĺ- **Firebase Storage:** Use the same bucket structure or consolidate into one. Image URLs include long tokens ÔÇö for the clone, use Firebase Storage SDK to generate signed URLs or make buckets public.
